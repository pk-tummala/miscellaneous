# miscellaneous

![Shell](https://img.shields.io/badge/shell-bash-1f425f.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

A collection of small, self-contained engineering utilities and quirks I've built
over the years — the kind of practical automation that quietly saves a team time
without needing a heavyweight framework behind it.

Each folder is independent: its own README, its own sample data, and where possible
a runnable demo so you can see the output without any special setup.

## Contents

| Project | What it does |
|---------|--------------|
| [`daily_job_status_automation/`](daily_job_status_automation/) | A pure-shell tool that queries DataStage master sequences and emails a RAG colour-coded daily status report. Runs in a self-contained demo mode out of the box. |

_More utilities will be added over time._

## Philosophy

When the requirements and boundaries of a problem are clearly known, simple
automation often beats a complex solution. These projects lean on that idea:
small, well-bounded tools that do one job reliably, every day.
